# duckpond

a duckpond for your terminal

```
                      _
            quack!  >|.|_
                     |___|
```

## usage
to visit the pond without installing:

```shell
python <(curl -s https://raw.githubusercontent.com/gsobell/duckpond/main/duckpond.py)
```

## features

### current
- water
- duck

### future
